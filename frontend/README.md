# Expense Tracker Application

## Overview
The Expense Tracker Application is a web-based tool designed to help users manage their expenses efficiently. Users can sign up, log in, and track their spending habits through a user-friendly interface. The application includes features such as expense tracking, reporting, and a chatbot assistant for enhanced user experience.

## Features
- **User Authentication**: Users can sign up, log in, and reset their passwords using OTP verification.
- **Expense Tracking**: Users can add, view, and analyze their expenses.
- **Monthly Reset**: Transactions reset every month, allowing for fresh analysis.
- **Reports**: Users can generate and download reports of their expenses.
- **Chatbot Assistant**: A chatbot is available to assist users with their queries related to expense tracking.

## Setup Instructions
1. Clone the repository to your local machine.
2. Open the `index.html` file in a web browser to access the application.
3. Follow the on-screen instructions to sign up or log in.

## Usage
- Navigate through the application using the provided links.
- Use the dashboard to track and manage your expenses.
- Access reports to analyze your spending habits.
- Utilize the chatbot for assistance with any queries.
